# Eugenia-Sirius
## Limitations
- You are only able to follow references to get to any elements
- Graphical Options can only be default
  - Except Colour
  - Labeling


 ## Basic use
 ### @sirius.diagram
 - name
 ### @sirius.node
 - name
 - container
 - label (default name)
 - color
 ### @sirius.container
 - name
 - container
 - label (default name)
 - color (TODO)
 ### @sirius.edge
This can work on both EClass and EReference
 - name
 - container
 - label (default No label)
 - source (EClass only)
 - target (EClass only)
